# Bedtime Buddies

A stuffed animal randomizer for bedtime — pick how many stuffies you want tonight, hit randomize, and let fate choose your crew.

**Live at: https://justinritchie.github.io/bedtime-buddies/**

## What It Does

Kids have too many stuffed animals and can't decide which ones get to sleep in the bed tonight. Bedtime Buddies solves this critical problem.

- Choose how many stuffies to bring to bed (1 to 23, or all of them)
- Tap **Randomize** to pick tonight's crew
- Watch them pop in one by one with a satisfying animation
- The full roster shows who made the cut and who's sitting this one out

## Tech

Single `index.html` file — no build step, no dependencies to install.

- **React 18** via CDN
- **Babel Standalone** for in-browser JSX transpilation
- **Custom SVG icons** for each stuffie
- CSS animations (pop-in reveals, floating title, twinkling stars background)
- Mobile-first responsive design with PWA meta tags

## Running Locally

Open `index.html` in a browser. That's it.
